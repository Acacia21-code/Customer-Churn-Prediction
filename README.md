# Customer Churn Prediction

## Project Overview

This project builds a machine learning model to predict customer churn — whether a customer will leave a service or not. It demonstrates a complete data science workflow including:

- Data loading and preprocessing
- Exploratory data analysis (EDA)
- Feature engineering and selection
- Model training using classification algorithms
- Model evaluation with accuracy, precision, recall, F1-score, and confusion matrix
- Insights to help reduce customer churn

This project is designed to be easy to understand and run, making it ideal for showcasing practical machine learning skills in a business context.

---

## Dataset

The dataset used contains customer information and their churn status. It typically includes features such as customer demographics, account information, service usage, and whether the customer churned (target variable).

> **Note:** Please replace the dataset path or download link in the code with your actual dataset location.
